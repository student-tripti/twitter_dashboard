## 📊 Twitter Analytics Dashboard (Power BI)

### 📌 Overview
Built an interactive dashboard to analyze social media performance using Power BI.

### 🎯 Features
- Likes, Retweets, Impressions tracking  
- Engagement rate analysis  
- Hashtag and URL clicks insights  
- Monthly and daily trends  
- Interactive filters
